- React js + Vite js

- Axios

## inicio sesión

- username: admin

## Guardar producto
### para guardar un producto necesitarás escribir en ciertos inputs de manera correcta 
### los datos, estos inputs y su valor son:
- connection type: directa
- storage system: interno
- condition: usado o nuevo
- owner: RF
